### 数据类型
`Unsigned char` :变量长度为1Byte(8位)
`Signed char `:变量长度为1Byte(8位)

`unsigned int `（0-65536）（16 bit）
`signed  int`（-32768--32767） （16 bit）
`unsigned long int`（0-4294967295）（32 bit）

### 中断服务程序

`函数返回类型 函数名（输入） interrupt n`

对应中断源的编号 n

| 编号  | 中断源     | 入口地址  |
| --- | ------- | ----- |
| 0   | 外部中断 0  | 0003H |
| 1   | 定时计数器 0 | 000BH |
| 2   | 外部中断 1  | 0013H |
| 3   | 定时计数器 1 | 001BH |
| 4   | 串行口中断   | 0023H |
